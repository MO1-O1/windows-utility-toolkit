# MO1 Windows Utility Toolkit

A lightweight Windows **Batch (.bat)** toolkit that combines common maintenance tasks into one simple menu.

## Collaboration / Credits
This project was created as a collaboration between:
- **MO1_1** — Project owner, direction, and features
- **ChatGPT** — Technical assistant (implementation support)

## Features
- Clean temporary files
- Run **SFC** (System File Checker)
- Run **DISM** (RestoreHealth)
- Restart network (Release/Renew/Flush DNS)
- Close common apps (Discord, Chrome)
- Show system information

## How to Use
1. Download or clone this repository
2. Right-click `toolkit.bat`
3. Select **Run as administrator**
4. Choose an option from the menu

## Requirements
- Windows 10 / Windows 11
- Administrator privileges for best results

## Disclaimer
Use at your own risk. Some actions may require admin permissions and can take time (e.g., DISM, SFC).
